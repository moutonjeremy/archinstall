Arch Linux Install

Automated installation scripts for Arch Linux

Quick Install

1 Base installation (from the Arch ISO)

curl -sLO https://moutonjeremy.github.io/archinstall/base.sh && bash base.sh

This script guides you through the complete installation: partitioning, formatting, base system installation, systemd-boot configuration, and user creation.

2 Download apps list and run installation (after first boot)

curl -sLO https://moutonjeremy.github.io/archinstall/apps.txt
curl -sLO https://moutonjeremy.github.io/archinstall/apps.sh && bash apps.sh

Edit apps.txt to customize the packages before running the script. You can also use your own file:

bash apps.sh /path/to/my-apps.txt

Features

base.sh - System installation

  • Interactive configuration (hostname, user, locale, keyboard, timezone)
  • UEFI mode required
  • Automatic partitioning (boot 1GB, swap 4GB, root)
  • Installation via pacstrap
  • systemd-boot bootloader
  • NetworkManager activation

apps.sh - Applications and configuration

  • Reads package list from apps.txt (local or URL)
  • yay installation (AUR helper)
  • Supports custom apps file via argument
  • Supports custom shell commands

apps.txt - Package list format

  • package - pacman package
  • [aur] package - AUR package
  • [cmd] command - custom shell command
  • Lines starting with # are comments

Detailed Usage

Warning: These scripts will format the disk and erase all data. Make sure you have a backup.

Prerequisites

  • Boot from the Arch Linux ISO
  • Active internet connection (ping archlinux.org)
  • UEFI mode required (for systemd-boot)

Steps

  1. Boot from the Arch Linux ISO
  2. Set keyboard layout: loadkeys us (or your layout)
  3. Connect to WiFi if needed: iwctl
  4. Run the base.sh script
  5. Follow the interactive instructions
  6. After reboot, log in and run apps.sh
